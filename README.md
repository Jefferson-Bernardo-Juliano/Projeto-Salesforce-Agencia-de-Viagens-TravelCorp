# Projeto Salesforce Agencia de Viagens TravelCorp
Este projeto foi desenvolvido como um ambiente prático de aprendizado, simulando o dia a dia de um profissional que utiliza o Salesforce para gerenciar processos de uma agência de viagens.  A aplicação, chamada TravelCorp, foi construída de forma incremental, com evolução contínua das funcionalidades — semelhante a um projeto real.
